# LabControl
## Introduction
Labcontrol is open source software designed to improve computer lab inventory management in addition to managing student loans. It's documentation will be avaliable in few days to explain in detail the objective and functionality of this micro SaaS prototype.

You can test the frontend following the instructions of the **instalation** section.

## Instalation
### To run LabControl in your device, follow this commands: 

*Clone this repository in your desired folder:*

``git clone https://github.com/eduardopozenatto/ppi-next-js.git``

*Navigate to the frontend folder:*

``cd ppi-next-js/frontend``

*Install pnpm package manager:*

``npm install -g pnpm``

*Install all pnpm dependencies:*

``pnpm install``

*Now, you can run this project in localhost using:*

``pnpm dev``
